# Contributing Guidelines

Thanks for helping grow this list! 🚀 Please read the guidelines below before opening a pull request.

## What we accept

We list **creative, unique, and original personal portfolios** built by their owner.

- ✅ Custom-built sites that show real creativity or craft.
- ❌ Templated, cloned, or drag-and-drop builder sites.
- ✅ New categories or improvements to the existing categorization are welcome.

## Before you submit

- **Search first.** Check the [README](README.md) and open pull requests so you don't submit a duplicate.
- **Check spelling and grammar** in your entry and description.
- **Keep descriptions short and simple.**

## How to add an entry

Add your site to the correct section of the [README](README.md), keeping the alphabetical/existing order intact. Use this exact format:

```html
- <img src="https://www.worldometers.info/img/flags/us-flag.gif" title="United States" height="14px" width="25px" style="border-radius:3px"> [Your Name](https://your-site.com/)
```

Formatting rules:

- **Flag** — include the owner's country flag using the same format as existing entries. Swap `us-flag.gif` and `title="United States"` for the correct country.
- **Name** — link your display name to the live site URL.
- **Titles / anchors** — no spaces; use an underscore (`_`) instead.

## Opening the pull request

- Make your PR title and description as clear and descriptive as possible.
- One site per pull request is preferred.

Thanks for your suggestion! 🎉
